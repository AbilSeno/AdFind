# AdFind
Mass admin login finder with multi-threading.
# Requirements
```
requests
```
# Installation
```
$ apt update && apt upgrade
$ pkg update
$ pkg install python2
$ git clone https://github.com/AbilSeno/AdFind
$ cd AdFind
$ pip install requests
$ python2 find.py https://domain.com
```
# Usage :
```
python2 find.py <domain>
```
